## 📌 Naming Conventions

- **Folder names** use `Snake_Case` for consistency
- **Numbered prefixes** (`1_`, `2_`, ...) maintain browse order
- **Subfolders** group books by specific topic or language
- **File names** preserve original book titles for easy identification

---

## 🛠️ Maintenance Guidelines

1. **Adding new books**: Place them in the appropriate subcategory folder. If no suitable folder exists, create a new one following the `Snake_Case` naming convention.
2. **New categories**: Add a new top-level folder with the next sequential number (e.g., `6_New_Category/`).
3. **Multiple formats**: Keep different formats (`.pdf`, `.epub`) of the same book side-by-side in the same folder.
